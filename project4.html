Twitetr | Web Application Showcase

The "Twitetr" project is a web service that allows users to write tweets, which are then spell-checked before being published. The text is analyzed using the LIBRIS spelling API, providing spelling suggestions. After reviewing the suggestions, users can approve and publish their tweets. This project integrates APIs such as the Twitter API for tweet management and the LIBRIS API for spelling correction, delivering an efficient and user-friendly experience for tweet composition and error handling.

Design Process

Initial Concept: The core idea was to build an easy-to-use platform for users to write and refine tweets, while ensuring accuracy and professionalism. The system interacts with external APIs (LIBRIS and Twitter) to provide seamless error detection and tweet publishing. We focused on designing a responsive, intuitive frontend, paired with a robust backend that handles API requests and data management.
Iteration 1: In the initial iteration, the primary focus was on developing a functional frontend that could capture tweet text, interact with the LIBRIS API for spelling correction, and send data to the Twitter API for publishing.
Iteration 2: The second iteration improved the UI/UX by implementing real-time spell-checking and suggesting corrections instantly as the user types their tweet. This was enhanced by clear error feedback and user guidance.
Iteration 3: The final iteration refined the integration with the APIs, allowing the backend to efficiently handle multiple API calls and send back JSON responses. We also focused on providing a clean, responsive UI for both desktop and mobile users.

My Contributions

  • Designed and developed the frontend interface using HTML, CSS, and JavaScript
  • Implemented real-time spelling corrections by integrating the LIBRIS API
  • Created user-friendly error handling features for a seamless tweet composition experience
  • Worked on backend logic to handle tweet publishing via the Twitter API
  • Ensured responsive design for both desktop and mobile devices

Outcome

The final product is a fully functional web application where users can compose, spell-check, and publish tweets with ease. Key features include real-time spelling suggestions from the LIBRIS API, an intuitive user interface for composing tweets, and seamless integration with the Twitter API for publishing. The application successfully streamlines the process of writing and sharing tweets while ensuring a professional and error-free output.
